# Tinfoil Core

Monorepo containing all trusted code and configurations for [Tinfoil](https://tinfoil.sh). All artifacts are attested to [Sigstore](https://www.sigstore.dev/).

## Structure

- **`platform/`** — Firmware code and platform configurations  
- **`system/`** — Linux kernel, NVIDIA drivers, and CVM image
- **`application/`** — Enclave code and tooling
